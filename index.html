第一週與 Android 基礎

  • 建立專案常見欄位：Name、Package、Minimum SDK、Language。
  • AndroidManifest.xml 宣告 Activity、權限、主題。
  • res 常見：layout、drawable、values。
  • Gradle 負責建構與依賴管理。

Kotlin 基礎

  • val 不可重設，var 可重設。
  • 可空型別加 ?，安全呼叫 ?.，!! 可能拋錯。
  • 條件可用 if-else / when。
  • 函式使用 fun，可帶參數、回傳值、預設值。

Ch2 介面設計與佈局

  • Linear / Frame / Constraint 常見佈局。
  • LinearLayout 未設 orientation 預設 horizontal。
  • ConstraintLayout 至少要水平與垂直約束。
  • 文字用 sp，尺寸間距用 dp。

Ch3 物件控制與事件監聽

  • 先 setContentView 再 findViewById。
  • 按鈕事件：setOnClickListener { ... }。
  • RadioButton 用 isChecked 判斷選項。
  • R 類別管理資源 ID。

Ch4 訊息提示元件

  • Toast：Toast.makeText(...).show()。
  • Snackbar 可用 setAction() 加互動按鈕。
  • AlertDialog：setTitle、setMessage、setItems、setSingleChoiceItems。
